# UI system tests

Системные UI-тесты реализованы на `Playwright`.

Что покрыто:

- эквивалентное разбиение для поиска, фильтрации и доступности флагов;
- анализ граничных значений для длины имени продукта, суммы БЖУ и размера порции блюда;
- системные сценарии создания продуктов и блюд через UI;
- автоматический расчёт КБЖУ, обработка макросов в названии блюда и запрет удаления продукта, который используется в блюде.

Запуск:

```bash
cd frontend
npm run test:ui
```
