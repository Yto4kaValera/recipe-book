# Backend Java

Отдельный backend на Java 17 и Spring Boot.

## Запуск

```bash
cd backend-java
mvn spring-boot:run
```

## Возможности

- CRUD для продуктов и блюд
- фильтрация, поиск и сортировка продуктов
- фильтрация и поиск блюд
- автоматический расчёт КБЖУ блюда
- макросы категорий в названии блюда
- проверка доступности флагов блюда
- запрет удаления продукта, если он используется в блюде
